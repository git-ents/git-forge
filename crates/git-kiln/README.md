# 🏺 `git-kiln`

*A ceramic-inspired toolkit for shaping Git-native data.*

> [!CAUTION]
> This project is in active development and has not yet been published to crates.io.
> Please file a [new issue] for any misbehaviors you find!

[new issue]: https://github.com/git-ents/git-forge/issues/new

## Overview

`git-kiln` is a planned companion tool for the `git-forge` workspace.
It will provide utilities for transforming, validating, and shaping data stored in Git's object database.

This crate is currently a placeholder — no subcommands are implemented yet.

## Installation

### CLI

```shell
cargo install --locked --git https://github.com/git-ents/git-forge.git git-kiln
```

If `~/.cargo/bin` is on your `PATH`:

```shell
git-kiln -h
```
